This package defines user-facing authentication policy.

Authenticator.PrincipalOutput

PrincipalOutput defines which principal (source.principal or request.auth.principal) that the authenticator will set if succeeds.

Name Description
UNSPECIFIED

Sentinel value to make sure the principal type must be set.
VALIDATE_ONLY

Does not set any principal attributes. Use this if you want the authenticator validating extra credentials without changing the principal outputs.
SOURCE_PRINCIPAL

Output principal to source.principal
REQUEST_AUTH_PRINCIPAL

Output principal to request.auth.principal

MutualTls.Mode

Defines the acceptable connection TLS mode.

Name Description
INVALID

Must never be set.
STRICT

Client cert must be presented, connection is in TLS.
PERMISSIVE

Connection can be either plain text or TLS, and client cert can be omitted.
